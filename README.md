## VSCodeの説明
- taskやlauch、ユーザスニペットなどの設定の保存は各フォルダの隠しファイル(.vscode)にある

## ショートキー
|キー (win)|機能|
|:--|:-:|
|F1 または Ctrl+Shift+p|コマンドパレットの表示|
|opt(Alt)+shift+F|コードのフォーマット|
|F12|定義へ移動|
|cmd(Ctrl)+1\|2\|3|タブの移動|
|opt(Alt)+上下矢印|上下にコードを移動|
|shift+opt(Alt)+上下矢印|上下にコードをコピー|
|cmd+x|行のコピー、または削除|
|cmd + shift + b|タスクの実行|

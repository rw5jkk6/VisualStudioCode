## gitの使い方(ターミナルでなく、vscodeのサイドバーで操作する)
- リポジトリの初期化(git init)
- 変更したファイルをステージング(git add)
  - \[+] アイコンを押す 
- ファイルのコミット(git commit)
  - コメントを入力して ctrl + enter 

## gitのコマンド
|コマンド|機能|
|:--|:--|
|git status|ファイルの状況を確認する|
|git diff|今のファイルとステージングされたファイルを比較する|

## リモートリポジトリの複製(Gitのclone)
